# Habit Tracker

A beginner-friendly habit tracker built using
vanilla HTML, CSS, and JavaScript.

Supports weekly tracking, streaks,
localStorage persistence, and responsive design.

## Features

- Add habits
- Edit habits
- Delete habits
- Weekly habit tracking
- Dynamic streak counter
- Week navigation
- Highlight current day
- localStorage persistence
- Responsive design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API
